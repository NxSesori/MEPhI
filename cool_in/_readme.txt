TYPE cool_in() - это функция типа TYPE (целочисленный, знаковый или беззнаковый, любого размера).
Функция использует библиотеку <conio.h>, доступную на oc Windows.
Функция считывает символы с клавиатуры. Все, что не является числом, будет проигнорировано. Также пользователь может стереть часть числа (backspace). 
Функция написана так, что пользователь ни при каких обстоятельствах не сможет вызвать переполнение перменной типа TYPE.
